# 必要な依存

- Node.js v16 以上推奨
- yarn (Node.js が入っている場合は`corepack enable`)

# 開発方法

```sh
yarn install
yarn dev # https://localhost:3000
```

- 注意: [拡張機能](https://chrome.google.com/webstore/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf)などでブラウザの CORS を無効にしてください
